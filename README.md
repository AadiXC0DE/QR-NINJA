## üöÄ Getting Started

1. **Clone the repository**:
   ```sh
   git clone https://github.com/AadiXC0DE/QR-NINJA.git
   ```

2. **Navigate to the directory**:
   ```sh
   cd QR-NINJA
   ```

3. **Install dependencies**:
   ```sh
   npm install
   ```

4. **Start the development server**:
   ```sh
   npm run dev
   ```

5. **Open your browser**: visit `http://localhost:3000`.
